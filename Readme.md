# Ramee Machine Test

The Tech stack i used in this test are

- React 18+
- Larvel10
- Mysql
- Xampp 3.3
- PHP 8.2

Time Taken: 2hrs 27min

How To Run Code

### React

```bash
cd client
npm install
npm run dev
```

The page will be open in `http://localhost:5173/`

### Laravel

```bash
cd server
cd example-app
```

- Migrate all tables

  ```bash
  php artisan migrate
  ```

- Run seeders

  ```bash
  php artisan db:seed
  ```

- Start server

  ```bash
  php artisan serve
  ```
